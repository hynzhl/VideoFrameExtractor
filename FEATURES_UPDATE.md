# 新增功能说明

## 更新日期
2026-04-14

## 新增配置项

### 1. 时间范围设置 ⏱️

现在可以指定从视频的哪个时间段提取帧:

#### 起始时间 (Start Time)
- **位置**: "时间范围设置" 区域
- **单位**: 秒 (支持小数,精确到0.01秒)
- **范围**: 0 - 86400 秒 (24小时)
- **默认值**: 0 (从视频开头开始)
- **说明**: 
  - 设置为 0 或留空表示从视频开头开始
  - 例如: 设置为 10.5 表示从第10.5秒开始提取

#### 终止时间 (End Time)
- **位置**: "时间范围设置" 区域
- **单位**: 秒 (支持小数,精确到0.01秒)
- **范围**: 0 - 86400 秒 (24小时)
- **默认值**: -1 (到视频结尾)
- **说明**:
  - 设置为 0 或留空表示提取到视频结尾
  - 例如: 设置为 60.0 表示在第60秒停止提取

**使用示例**:
- 起始时间: 10, 终止时间: 30 → 只提取第10秒到第30秒之间的帧
- 起始时间: 0, 终止时间: 0 → 提取整个视频

---

### 2. 输出分辨率设置 📐

可以自定义输出图片的分辨率:

#### 宽度 (Width)
- **位置**: "输出分辨率设置" 区域
- **单位**: 像素
- **范围**: 0 - 7680
- **默认值**: 0 (保持原始宽度)
- **说明**: 设置为0表示保持原始视频宽度

#### 高度 (Height)
- **位置**: "输出分辨率设置" 区域
- **单位**: 像素
- **范围**: 0 - 4320
- **默认值**: 0 (保持原始高度)
- **说明**: 设置为0表示保持原始视频高度

---

### 3. 宽高比设置 🖼️

可以指定输出图片的宽高比:

#### 可选模式:
1. **保持原始** (默认)
   - 不改变视频的原始宽高比
   
2. **16:9**
   - 宽屏比例,适合现代显示器
   - 例如: 1920x1080, 1280x720
   
3. **4:3**
   - 传统电视比例
   - 例如: 1024x768, 800x600
   
4. **1:1 (正方形)**
   - 正方形图片
   - 例如: 512x512, 1024x1024
   
5. **自定义**
   - 使用设置的宽度和高度值

**说明**:
- 如果设置了宽高比但未设置具体尺寸,会保持原始分辨率但调整比例
- 如果同时设置了尺寸和宽高比,会按指定尺寸和比例输出
- 图片会自动填充黑边以保持指定的宽高比

## FFmpeg命令构建逻辑

更新后的FFmpeg命令结构:

```bash
ffmpeg.exe 
  [-ss <起始时间>]           # 如果设置了起始时间
  -i <视频文件>
  [-to <终止时间>]           # 如果设置了终止时间
  -vf "fps=1/<间隔>[,scale=...]"  # 抽帧+缩放滤镜
  -q:v 2
  <输出目录>/frame_%06d.png
  -y
```

### 视频滤镜链 (Video Filter Chain)

滤镜按以下顺序应用:
1. **fps** - 抽帧滤镜 (必选)
2. **scale/pad** - 缩放和填充滤镜 (如果设置了分辨率或宽高比)

**示例**:
```bash
# 每2秒一帧,缩放到1920x1080,16:9比例
-vf "fps=1/2,scale=1920:1080:force_original_aspect_ratio=decrease,pad=1920:1080:(ow-iw)/2:(oh-ih)/2"

# 每5秒一帧,保持原始分辨率
-vf "fps=1/5"

# 每秒一帧,缩放到512x512正方形
-vf "fps=1/1,scale=512:512:force_original_aspect_ratio=decrease,pad=512:512:(ow-iw)/2:(oh-ih)/2"
```

## 使用场景示例

### 场景1: 提取视频精彩片段
```
起始时间: 120    (从第2分钟开始)
终止时间: 180    (到第3分钟结束)
时间间隔: 1      (每秒一帧)
分辨率: 0x0      (保持原始)
宽高比: 保持原始
```
结果: 提取第2-3分钟的60帧图片

### 场景2: 生成视频缩略图
```
起始时间: 0
终止时间: 0      (整个视频)
时间间隔: 30     (每30秒一帧)
分辨率: 320x180
宽高比: 16:9
```
结果: 生成小尺寸缩略图,适合预览

### 场景3: 制作社交媒体内容
```
起始时间: 0
终止时间: 60     (前60秒)
时间间隔: 2
分辨率: 1080x1080
宽高比: 1:1 (正方形)
```
结果: 正方形图片,适合Instagram等平台

### 场景4: 高清存档
```
起始时间: 0
终止时间: 0
时间间隔: 5
分辨率: 1920x1080
宽高比: 16:9
```
结果: 高清16:9图片,适合存档

## 技术实现细节

### C++ 类变更

#### 新增属性 (Q_PROPERTY):
```cpp
Q_PROPERTY(double startTime READ startTime WRITE setStartTime NOTIFY startTimeChanged)
Q_PROPERTY(double endTime READ endTime WRITE setEndTime NOTIFY endTimeChanged)
Q_PROPERTY(int outputWidth READ outputWidth WRITE setOutputWidth NOTIFY outputWidthChanged)
Q_PROPERTY(int outputHeight READ outputHeight WRITE setOutputHeight NOTIFY outputHeightChanged)
Q_PROPERTY(int aspectRatio READ aspectRatio WRITE setAspectRatio NOTIFY aspectRatioChanged)
```

#### 新增方法:
```cpp
QString buildFFmpegFilterString() const;  // 构建FFmpeg滤镜字符串
```

#### 新增成员变量:
```cpp
double m_startTime;     // 起始时间(秒), -1表示从开头
double m_endTime;       // 终止时间(秒), -1表示到结尾
int m_outputWidth;      // 输出宽度
int m_outputHeight;     // 输出高度
int m_aspectRatio;      // 宽高比模式 (0-4)
```

### QML 组件变更

#### 新增控件:
- `DoubleSpinBox` x2 - 起始/终止时间
- `SpinBox` x2 - 宽度/高度
- `ComboBox` x1 - 宽高比选择
- `GridLayout` - 用于更复杂的布局

#### 布局改进:
- 添加 `ScrollView` 支持滚动
- 使用 `GridLayout` 替代部分 `RowLayout`
- 增加更多提示文本

## 已知限制

1. **进度显示**: 当使用起始时间时,进度百分比可能不够准确
2. **性能**: 高分辨率输出会增加处理时间
3. **内存**: 大尺寸图片会占用更多内存
4. **宽高比填充**: 使用pad滤镜添加黑边,不是裁剪
